# Proton

Proton is a simple text editor, soon-to-be IDE.

![Preview](https://imgur.com/AmzmMmn.png)

## Compile / Install
Use `flatpak` or `meson` (ninja) to install or compile Proton.

## Features worth sharing
- Integrated terminal
- Editorconfig plugin
